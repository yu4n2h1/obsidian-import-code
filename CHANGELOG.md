# 更新日志

## v1.1.0 (2026-05-26)

### 新增

- **本地文件系统源** — 远程源别名新增 "Local Directory" 服务类型，支持将本地任意目录配置为代码源，通过 `![[alias:path/file.ext]]` 引入外部库代码
- **设置数据自动迁移** — 从旧版 `remoteServices` 格式自动迁移到新版 `remoteSources` 格式，无需手动重新配置

### 优化

- **设置界面重构** — 远程源别名改为卡片式布局，别名支持直接重命名（含冲突检测），删除按钮增加警告样式，新增按钮增加说明文字
- **代码展示界面** — 修复行高亮渲染时高亮行之间出现多余空行的问题，显示更紧凑
- **扩展名过滤覆盖远程路径** — 远程/别名路径现在也遵循文件扩展名过滤规则
- **设置即时生效** — 开启/关闭代码嵌入开关后自动刷新已打开的 Markdown 视图，无需重新打开笔记

---

## v1.0.0 (2026-05-20)

- 正式版首发，支持本地/远程代码嵌入、符号提取、行范围/高亮、远程源别名、双模式（阅读/实时预览）等核心功能
